# NAND-Gate-Using-BJTs

## ✍ Overview
This project demonstrates the design and implementation of a NAND logic gate using discrete BJT transistors (BC547).
The circuit operates with a 5V power supply and illustrates how basic electronic components can be used to build fundamental digital logic gates.

The main objective of this project is to provide a practical understanding of transistor switching behavior and how it can be applied to implement digital logic functions.

##  📌 Project Objective
The goal of this project is to:

- Implement a NAND gate using BJT transistors
- Understand transistor switching operation
- Demonstrate how digital logic circuits can be built using discrete components
- Provide a hardware-level understanding of logic gates

##  🔍 How a NAND Gate Works
![NAND Gate](https://www.build-electronic-circuits.com/wp-content/uploads/2022/09/Truth-table-NAND-gate.png)

The output becomes LOW only when both inputs are HIGH.

Because of this property, the NAND gate is considered a Universal Logic Gate, meaning any digital circuit can be built using only NAND gates.

## Components Used
- BC547 Transistor (2)
- RES (6)
- led (1)
- DC Input Voltage Module to Breadboard (2)
- Button (2)

## 💯 Circuit Operation
The transistors in the circuit operate in switching mode.

When both inputs are HIGH, both transistors conduct → output becomes LOW.
If any input is LOW, one transistor stops conducting → output becomes HIGH.

This switching behavior replicates the NAND logic function.
